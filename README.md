MSA App Demo — GitHub Pages Deployment Guide
Files
msa-app-demo.html → Self-contained demo page (no build needed!)
msa-mockup.jsx → React component (for developers)
Deploy to GitHub Pages in 5 minutes
Step 1 — Create GitHub repo
Go to github.com → New repository
Name it: msa-app-demo (or moresuccessacademy-demo)
Set to Public
Click "Create repository"
Step 2 — Upload the file
Click "uploading an existing file"
Upload msa-app-demo.html
RENAME it to index.html (GitHub Pages serves index.html)
Commit changes
Step 3 — Enable GitHub Pages
Go to repo Settings → Pages
Source: "Deploy from branch"
Branch: main / root
Click Save
Step 4 — Your live URL
After ~2 minutes:
https://YOUR-USERNAME.github.io/msa-app-demo/
Embed on moresuccessacademy.com.ng
Add this HTML snippet to any page on your website:
<!-- MSA App Demo Embed -->

  width="100%"
  height="800px"
